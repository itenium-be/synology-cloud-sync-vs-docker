Synology CloudSync vs Docker
============================

Sync files generated by a docker job with Synology CloudSync.

## Configure .env

```ini
CRON="* * * * *"

# The synology dir that is being synced with CloudSync
SYNC_DIR=~/sync-dir
```

## And start

Make `create-file.sh` do something.

```sh
docker-compose up -d --build
```
